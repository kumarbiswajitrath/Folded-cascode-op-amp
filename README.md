# Folded-cascode-op-amp
## Overview
Designed a **Folded Cascode Operational Amplifier** in **Cadence Schematic Composer (SCL)**, achieving **60 dB AC Gain** and **10 MHz Unity Gain Bandwidth (UGB)**. The design was validated using **DC, AC, and transient analyses** to evaluate gain, frequency response, and stability.

## Features
- Folded Cascode Op-Amp architecture
- 60 dB AC Gain
- 10 MHz Unity Gain Bandwidth (UGB)
- Designed using Cadence SCL
- DC, AC, and Transient simulations
- Optimized transistor sizing and biasing
- Stable frequency response
